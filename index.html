About

Light weight, Easy to learn JavaScript Templating Engine that can work on back-end with ExpressJs as well as on front-end. Perthite is very easy to learn if you have a JavaScript background bacause all does is to embed JavaScript code into your HTML code.

Why use Perthite ?

Perthite is lightweight thus has a shallow learning curve. Another thing is Perthite is a simple project to start contributing to, you don't need to be a JavaScript expert all you need is good background of JavaScript. You may also contribute by writting online tutorials and articles about Perthite or improving the documentation, that will be appriciated.

Installation

First thing is to make sure that you have NodeJS installed in your computer. If you don't have NodeJs installed you can download it here. If you have node installed, go to your termnal and run > npm i perthite

Usage With ExpressJs

JavaScript

const Express = require("express");
const app = Express();
const Perthite = require("Perthite");

app.engine('html', Perthite.engine);
app.set('views', './views');
app.set('view engine', 'html');

app.get('/', (req, res) => {
res.render("main" , { user: "Hello World" }); } );

app.listen(3000, (err) => {
return err && console.err(err) || console.log("server started on port 3000") }) Template

Calling functions on a Template

JavaScript

const Express = require("express");
const app = Express();
const Perthite = require("Perthite");

app.engine('html', Perthite.engine);
app.set('views', './views');
app.set('view engine', 'html');

app.get('/', (req, res) => {
res.render("main" , {
concat: (arg1, arg2)=>{ return arg1 + " " + arg2 }
});

app.listen(3000, (err) => {
return err && console.err(err) ||
console.log("server started on port 3000")
}) Template
