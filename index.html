Pottassium js

Light weight, Easy to learn JavaScript Templating Engine that can work on back-end with ExpressJs as well as on front-end. Potassium is very easy to learn if you have a JavaScript background bacause all does is to embed JavaScript code into your HTML code.

Getting Sartted

First thing is to make sure that you have Nodejs installed in your computer. If you don't have NodeJs installed you can download it here. For now you can download Potassium project from its github repository and start using right away.

Usage With ExpressJs

JavaScript

const Express = require("express");
const app = Express();
const photon = require("potassium");

app.engine('html', photon.engine);
app.set('views', './views');
app.set('view engine', 'html');

app.get('/', (req, res) => {
res.render("main" , { user: "Hello World" }); } );

app.listen(3000, (err) => {
return err && console.err(err) || console.log("server started on port 3000") }) Template
